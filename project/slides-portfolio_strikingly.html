SOLUTION - PRODUCT FILTER

Problem:

For e-commerce website, buyers can't filter products. 

 

Solution:

1. Sellers can activate Product Filter EFFICIENTLY.

2. Sellers can choose which dimensions can be filtered EASILY.

3. Buyers can filter products by the dimensions that sellers selected. 

Buyer Side:
Can filter products by product dimensions, and clear all selected options by one click.

METHODOLOGY SUMMARY

1. Empathy

  • Analyze USER MENTAL to evaluate where the function should be.

  • Support the same function that user needs in different scenes.

2. Interaction

  • Consider multiple styles on website editing
  • Make the function VISIBLE and use natural interaction mode.

(e.g. click activate button in Categories page means click checkbox in layout setting controller）

(e.g. what if change the background color or background image）
